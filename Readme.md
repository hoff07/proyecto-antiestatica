# Proyecto Antiestatica - Sistema de Vinilos
Tienda especializada en discos de vinilo
Desarrollado por: Hoffman Elizalde
Contacto: hoffman-elizalde@upc.edu.co
Nueva caracteristica: catalogo de vinilos por genero

## Como clonar el proyecto
git clone https://github.com/hoff07/proyecto-antiestatica.git

## Como contribuir
1. Crea una rama nueva: git checkout -b nombre-rama
2. Haz tus cambios y commits
3. Sube la rama: git push origin nombre-rama
4. Abre un Pull Request en GitHub

## Como ejecutar el proyecto
1. Clona el repositorio
2. Navega a la carpeta: cd proyecto-antiestatica
3. Revisa el archivo Readme.md
